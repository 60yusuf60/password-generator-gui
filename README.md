# Python GUI Password Generator 🔐

A simple Python GUI application that generates strong random passwords.

## Features
- Generate secure passwords with letters, numbers, and symbols
- Custom password length
- Easy to use graphical interface (Tkinter)

## Installation

Clone the repository:

git clone https://github.com/60yusuf60/password-generator-gui.git

## Usage

Run the script:

python main.py

Enter desired password length and click "Generate Password".

## Technologies Used

- Python

- Tkinter
